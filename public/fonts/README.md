# Open Sauce Sans Font Files

This directory should contain the Open Sauce Sans font files in WOFF2 format.

## Required Files:

1. **OpenSauceSans-Regular.woff2** (weight: 400)
2. **OpenSauceSans-Medium.woff2** (weight: 500)
3. **OpenSauceSans-SemiBold.woff2** (weight: 600)
4. **OpenSauceSans-Bold.woff2** (weight: 700)

## Download Instructions:

You can download Open Sauce Sans from:
- [Creative Market](https://creativemarket.com/SourceFoundry/2804325-Open-Sauce-Sans)
- [GitHub](https://github.com/marcologous/Open-Sauce-Fonts) (if available)
- Or use a similar font like "Inter" or "Open Sans" as an alternative

## Installation:

1. Download the font files in WOFF2 format
2. Place them in this directory (`public/fonts/`)
3. The font-face declarations are already configured in `app/styles/globals.css`

## Alternative:

If you cannot obtain Open Sauce Sans, you can use Google Fonts alternatives:
- Inter (very similar)
- Open Sans
- Source Sans Pro

Update the font-family in globals.css to use the alternative font.
